# Books Dataset Analysis – Excel & Power BI

## Project Overview

This project focuses on analysing a books dataset to identify patterns and trends using data cleaning and visualization techniques.

## Tools Used

* Microsoft Excel
* Power BI

## Project Workflow

* Cleaned and preprocessed raw books data using Microsoft Excel
* Handled missing values, duplicate records, and data inconsistencies
* Prepared clean datasets for analysis and reporting
* Created interactive dashboards in Power BI using charts, filters, and slicers
* Visualised trends and patterns to support data-driven insights

## Files in this Repository

* Books_Dataset_Cleaned.xlsx – Cleaned dataset
* Books_PowerBI_Dashboard.pbix – Power BI dashboard file

## Key Insights

* Distribution of books across categories
* Trend analysis based on available data fields
* Interactive filtering for better data understanding
